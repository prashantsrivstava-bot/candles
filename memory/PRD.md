# Soil & Scent - Landing Page PRD

## Original Problem Statement
Create a cool and super landing page for "Soil and Scent" candle business with WhatsApp contact (8789028909), Instagram link (@soilandscent), and pre-filled WhatsApp message.

## User Personas
- **Primary**: Home decor enthusiasts looking for artisan candles
- **Secondary**: Gift shoppers seeking unique handcrafted items

## Core Requirements
- Brand: Soil & Scent (artisan candle business)
- WhatsApp: 8789028909
- Instagram: @soilandscent
- Pre-filled message: "Hi! I'm interested in your candles"

## Architecture
- **Frontend**: React with Tailwind CSS + Framer Motion
- **Design**: Organic & Earthy theme (light mode)
- **Typography**: Cormorant Garamond (headings) + Manrope (body)
- **Colors**: Warm terracotta (#C05C3D), sage green (#8B9A8B), cream (#FDFBF7)
